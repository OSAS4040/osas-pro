# ADR-0001: وضع واجهة الورشة السريع (Staff compact UI)

- **الحالة:** مقبول
- **التاريخ:** 2026-04-06
- **السياق:** أثناء الذروة تحتاج الشاشات الميدانية وقتاً أقل للتنقّل وأقل عناصر ثانوية دون إعادة بناء تطبيق جوال.
- **القرار:** تفعيل وضع اختياري يخفّض العناصر الثانوية في `/work-orders` عبر حالة Pinia + `localStorage`، مع افتراض بناء اختياري `VITE_STAFF_COMPACT_UI=true`.
- **البدائل:** تطبيق منفصل للفني فقط (أبطأ للتسليم)؛ إخفاء دائم للعناصر (يضر المستخدمين الإداريين).
- **العواقب:** يجب اختبار يدوي لتدفق «تحديث الحالة» والتوقيع؛ أي ميزة جديدة في أوامر العمل يجب أن تراعي `staffUi.compactMode`.
