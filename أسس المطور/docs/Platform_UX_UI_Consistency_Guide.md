# Platform UX/UI Consistency Guide

دليل اتساق تجربة الاستخدام والواجهات لمشروع Vue 3 + Tailwind (frontend). **WAVE 0:** مبادئ وقائمة تدقيق؛ التنفيذ الشامل في Wave 14 مع احترام كل موجة سابقة.

---

## 1. المبادئ (مستخرجة من متطلبات المنتج)

1. **مظهر مؤسسي حديث** — رصانة ERP/SaaS، بدون مبالغة بصرية تشتت المشغّل.  
2. **RTL ممتاز** — اتجاه النص، المحاذاة، الأيقونات، الجداول، النماذج متعددة الأعمدة.  
3. **اتساق** — ألوان الحالات، المسافات، حجم الخط، البطاقات، الجداول، الأزرار الأساسية/الثانوية.  
4. **حالات التحميل** — skeletons أو مؤشرات واضحة؛ تجنب الطفو المفاجئ للمحتوى.  
5. **حالات الفراغ** — empty states مع خطوة تالية واضحة.  
6. **الأخطاء** — رسائل قابلة للفهم، بدون تسريب تقني للمستخدم النهائي؛ تتبع تقني في الخلفية.  
7. **تعدد الأدوار** — لوحات و quick actions حسب الصلاحيات؛ إخفاء ما لا يُسمح به أفضل من تعطيل غامض.  
8. **تدفقات طويلة** — تقليل الاحتكاك، مسودات حيث يلزم، تقدم واضح (خطوات).  
9. **الطباعة والتصدير** — اتساق بين list / show / create / edit / print / export.  
10. **Mobile-friendly** — للحدود الإدارية الأساسية (قوائم، موافقات طارئة، عرض تذكرة).

---

## 2. مصفوفة الشاشات الحرجة (أولوية تدقيق)

| المنطقة | مسارات Vue (أمثلة من `router/index.ts`) | ملاحظة |
|---------|------------------------------------------|--------|
| الدخول والهاتف | `login`, `platform-login`, `phone-*` | Wave 1 — توحيد الهوية البصرية |
| POS | `pos` | أداء ووضوح سعر وضريبة |
| الفواتير | `invoices`, `invoices.show` | حالات الدفع والطباعة |
| المحفظة / الطلبات | إن وُجدت في الإعدادات أو الفواتير | وضوح الرصيد |
| التقارير | `reports`, `business-intelligence` | Wave 13 — عدم ازدحام غير مبرر |
| الإعدادات | `settings`, `settings.integrations` | أسرار وتلميحات أمان |

---

## 3. قائمة تدقيق Wave 14 (مرجعية)

- [ ] Typography — مقاييس خط موحدة عبر `layouts` والمكوّنات المشتركة  
- [ ] Spacing scale — 4/8/12/16 متسقة  
- [ ] الجداول — رؤوس ثابتة حيث يلزم، تمرير أفقي على الجوال  
- [ ] النماذج — تسميات، أخطاء تحقق مرتبطة بالحقول  
- [ ] الأزرار — primary واحد لكل لوحة مرئية رئيسية  
- [ ] Badges للحالات — نفس الألوان لنفس المعنى عبر الوحدات  
- [ ] Dark mode — إن مفعّلاً، تدقيق تباين  
- [ ] الطباعة — `@media print` للفواتير والتقارير الحرجة
