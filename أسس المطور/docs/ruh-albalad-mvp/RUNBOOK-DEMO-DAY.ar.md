# Runbook نهائي - Commission Ready

## 1) التشغيل

1. وصلي مزود الطاقة `5V/8A`.
2. تأكدي من `GND` المشترك بين جميع المكونات.
3. ارفعي الملف المعتمد فقط: `ruh_albalad_mvp_demo.ino`.
4. تأكدي أن الإعدادات هي:
   - `COMMISSION_READY_PRESET = true`
   - `BACKUP_SAFE_PRESET = false`
   - `ENABLE_TEST_MODE = false`
3. شغلي النظام وانتظري رسائل الإقلاع في Serial:
   - `[DEMO] BOOT`
   - `[DEMO] DF: ...`
   - `[DEMO] SD: ...`
   - `[DEMO] STATE: IDLE`

## 2) الفحص السريع (60 ثانية)

1. اضغطي `START` -> يجب الانتقال لـ `INTRO`.
2. في `PICK_SCENE` اضغطي `A` -> يجب الدخول لـ `SCENE_A`.
3. تأكدي من وصول النظام إلى `OUTRO` ثم العودة `IDLE`.
4. كرري الدورة 3 مرات متتالية قبل العرض الرسمي.

## 3) بدء العرض أمام اللجنة

- أفضل وضع: اتركي العرض على `IDLE` الجذاب.
- عند بداية الحديث اضغطي `START` (أو اتركي auto-start يعمل).
- لا تغيري أي إعداد أثناء العرض.

## 4) إعادة التشغيل عند الحاجة

- إذا توقف الصوت فقط: كملي العرض (Safe Demo Mode) ثم أعيدي التشغيل بعد الجولة.
- إذا لم يبدأ العرض:
  1) اضغطي `START`
  2) أو انتظري Auto-Start بعد 4 ثوان
- إذا توقف الحساس:
  1) استمري بزر `START` (الحساس ليس شرطا)
  2) بعد الجولة يمكن التحويل إلى `BACKUP_SAFE_PRESET = true`
- إذا حدث تجمد كامل:
  1) افصلي الطاقة 3 ثواني
  2) أعيدي التوصيل
  3) تحققي من `STATE: IDLE`

## 5) نسخة Backup الفورية (عند الطوارئ)

- استخدمي نفس الملف `ruh_albalad_mvp_demo.ino`.
- غيّري فقط:
  - `BACKUP_SAFE_PRESET = true`
- ثم ارفعي الكود.
- هذا الوضع يخفض الحمل (سطوع/صوت/حساس) لرفع الاستقرار.

## 6) قاعدة الانضباط قبل أي تطوير لاحق

لا انتقال لأي تحسين إضافي قبل نجاح:
1. تشغيل كل قطعة منفصلة
2. دورة كاملة بدون أخطاء
3. نسخة Demo تعمل 3 مرات متتالية بثبات
