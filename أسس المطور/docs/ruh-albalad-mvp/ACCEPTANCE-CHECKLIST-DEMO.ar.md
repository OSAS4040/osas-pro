# Acceptance Checklist - Demo اللجنة

## قبل الحضور (فحص تجهيز)

- [ ] تم رفع `ruh_albalad_mvp_demo.ino` بنجاح.
- [ ] `ENABLE_TEST_MODE = true` تم تشغيله مرة واحدة وفحص الأجزاء.
- [ ] `ENABLE_TEST_MODE = false` تم رفع النسخة النهائية للعرض.
- [ ] `COMMISSION_READY_PRESET = true` و `BACKUP_SAFE_PRESET = false`.

## اختبار الأزرار

- [ ] زر `START` يبدأ العرض فورا.
- [ ] زر `A` يعمل في مرحلة اختيار السيناريو.
- [ ] زر `B` يعمل في مرحلة اختيار السيناريو.

## اختبار LED

- [ ] الإضاءة تظهر بوضوح في `IDLE`.
- [ ] تأثير `INTRO` قوي وسريع.
- [ ] تأثير `SCENE_A` واضح ومبهر.
- [ ] `OUTRO` واضح وينتهي بسلاسة.

## اختبار DFPlayer / microSD

- [ ] تشغيل `0001.mp3` عند البداية.
- [ ] تشغيل `0002.mp3` أو `0003.mp3` حسب السيناريو.
- [ ] تشغيل `0004.mp3` في الخاتمة.
- [ ] عند فشل الصوت، العرض يستمر بصريا (Safe Demo Mode).

## اختبار الحساس

- [ ] الحساس يطلق البداية عند الاقتراب (إن كان مستخدما).
- [ ] عند تعطل الحساس يمكن البدء بزر `START` بدون أي مشكلة.

## اختبار دورة الحالات الكاملة

- [ ] `IDLE -> INTRO -> PICK_SCENE -> SCENE -> OUTRO -> IDLE`
- [ ] دورة كاملة ناجحة مرتين متتاليتين.
- [ ] دورة كاملة ناجحة 3 مرات متتالية (معيار اعتماد نهائي).

## اختبار الطاقة والثبات

- [ ] لا توجد إعادة تشغيل مفاجئة.
- [ ] لا يوجد وميض غير طبيعي.
- [ ] لا يوجد انقطاع صوت متكرر ناتج عن هبوط جهد.
- [ ] مزود الطاقة الخارجي 5V/8A موصل مع GND مشترك.

## نتيجة الاعتماد النهائي
