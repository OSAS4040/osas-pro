خادم الواجهة (Vite) غير متاح على المنفذ المتوقع

nginx يوجّه الجذر / إلى حاوية التطوير. إذا كانت الحاوية متوقفة أو المنفذ مختلفاً، يظهر هذا الإشعار بدل 502 الخام.

محلياً (Docker):

  • تأكد أن خدمة frontend تعمل: docker compose up -d frontend
  • أو افتح الواجهة مباشرة: http://127.0.0.1:5173
  • لواجهة مبنية للإنتاج، اضبط nginx لخدمة dist بدل البروكسي إلى Vite.

API: /api/v1/health
