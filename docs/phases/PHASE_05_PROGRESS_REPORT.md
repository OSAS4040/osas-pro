# تقرير تقدم المرحلة 5 — حوكمة وصلاحيات دقيقة

**حالة الوثيقة:** **تقدم** — مصفوفة الصلاحيات تتوسع مع كل ميزة جديدة.

## 1. الهدف

التأكد من أن الموارد الحساسة (مثل بنود الكتالوج التعاقدي) لا تعتمد على صلاحيات عامة خاطئة، وأن الرفض واضح عند غياب الإذن.

## 2. قائمة تحقق

| البند | الملاحظات |
|--------|------------|
| صلاحيات بنود الخدمة التعاقدية | `ContractServiceItemPermissionsTest` |
| مراجعة `config/permissions.php` عند إضافة endpoints | عملية يدوية |

## 3. الاختبارات

- مجموعة PHPUnit: **`phase5`**
- الأمر: `composer test:phase5`
- **واجهة (Vitest):** `cd frontend && npm run test:phase5` — `portalAccess` وتكافؤ `localeParity.teamOrg`.

## 4. فجوات

- تغطية أدوار إضافية (مثلاً محاسب مقيد) عند تعريف صلاحيات أدق في المنتج.
