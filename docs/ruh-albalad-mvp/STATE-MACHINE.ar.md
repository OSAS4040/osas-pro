# منطق الحالات (State Machine) - روح البلد MVP

## الحالات

1. `IDLE`
   - إضاءة خافتة ثابتة.
   - انتظار زر البدء أو إشارة الحساس.

2. `INTRO`
   - تشغيل ملف صوت افتتاحي.
   - تأثير ضوئي قصير لجذب الانتباه.
   - انتقال تلقائي إلى اختيار السيناريو.

3. `AWAIT_SCENARIO`
   - انتظار زر `A` أو `B`.
   - مهلة اختيار (مثال 10 ثوان).
   - عند انتهاء المهلة: اختيار افتراضي (`A`).

4. `SCENARIO_A (ازدهار)`
   - إضاءة دافئة ومضيئة.
   - صوت مرتبط بالازدهار.
   - مدة عرض محددة (30-45 ثانية).

5. `SCENARIO_B (إهمال)`
   - إضاءة باردة/خافتة.
   - صوت يوضح أثر الإهمال.
   - مدة عرض محددة (30-45 ثانية).

6. `OUTRO`
   - رسالة ختامية قصيرة.
   - Fade Out للإضاءة.
   - العودة إلى `IDLE`.

## شروط الانتقال

- `IDLE -> INTRO` عند `START` أو `Proximity`.
- `INTRO -> AWAIT_SCENARIO` بعد انتهاء الافتتاح.
- `AWAIT_SCENARIO -> SCENARIO_A/B` حسب اختيار الزائر.
- `SCENARIO_* -> OUTRO` بعد انتهاء مدة المشهد.
- `OUTRO -> IDLE` بعد انتهاء الرسالة.

## قواعد ثبات التشغيل

- يمنع استقبال مدخلات جديدة أثناء تشغيل مشهد نشط.
- أي خطأ صوتي لا يوقف الإضاءة (فصل منطقي بين الوحدتين).
- دعم `Watchdog` اختياري لاحقا إذا زاد تعقيد النسخة.
