# الذكاء الاصطناعي والتحليلات والتوصيات

## مكونات الذكاء
- Signals.
- Incident Candidates.
- Incidents.
- Decision Log.
- Controlled Actions.
- Guided Workflows.

## التسلسل
Collect -> Normalize -> Detect -> Correlate -> Score -> Dedupe -> Explain -> Prioritize.

## الحدود
- الذكاء داعم للقرار وليس بديلًا للحوكمة البشرية.
- التنفيذ المقيد يتطلب صلاحيات منصة.
