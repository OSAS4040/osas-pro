# مصفوفة القرارات والإجراءات

| القرار | من يملكه | الشروط | النتيجة |
|---|---|---|---|
| اعتماد طلب شحن | مراجع مالي | صلاحية review + مستندات مكتملة | رصيد مضاف + أثر مالي |
| رفض طلب شحن | مراجع مالي | نقص/عدم مطابقة | إغلاق بالرفض |
| إصدار فاتورة مبيعات للعميل (تلقائي) | محرك الفوترة التعاقدي | أمر عمل مكتمل + توثيق + تعاقد عميل صالح | فاتورة مبيعات عميل + أثر إيراد |
| إصدار فاتورة مشتريات للمزود (تلقائي) | محرك الفوترة التعاقدي | أمر عمل مكتمل + توثيق + تعاقد مزود صالح | فاتورة مشتريات مزود + أثر تكلفة |
| منع إرسال فاتورة المزود للعميل | سياسة المنصة | المسار القياسي للفوترة | حجب كامل لفاتورة المزود عن واجهة العميل |
| منع إصدار فاتورة/عرض سعر من العميل | سياسة الصلاحيات | جميع حسابات customer | العميل يطّلع فقط على الفواتير الصادرة عليه |
| تصعيد incident | مشغل منصة | incident تحت المراجعة + صلاحية | انتقال حالة + إشعار |
| إغلاق incident | مشغل منصة | مكتمل المعالجة + صلاحية close | إغلاق نهائي + trace |
